# Plan: Subida de archivos locales para Banners y Productos

## Objetivo
1. **Banners**: Quitar campo URL, dejar solo subida de archivo local
2. **Productos**: Cambiar de "Seleccionar de Galería" a subida de archivo local

## Archivo a modificar
`js/admin.js`

---

## Cambio 1: Banners - Quitar campo URL

### `editBanner()` (línea 484)
**Cambiar el modal de:**
```html
<input type="file" id="banFile" accept="image/*">
<input type="url" class="form-control" id="banUrl" placeholder="https://...">
```

**A:**
```html
<input type="file" id="banFile" accept="image/*">
```

Y en el handler async, cambiar:
```javascript
let url = document.getElementById('banUrl').value.trim();
const fileInput = document.getElementById('banFile');
if (fileInput.files && fileInput.files[0]) {
  url = await readFileAsDataURL(fileInput.files[0]);
}
```

**A:**
```javascript
const fileInput = document.getElementById('banFile');
let url = '';
if (fileInput.files && fileInput.files[0]) {
  url = await readFileAsDataURL(fileInput.files[0]);
}
```

### `addBannerBtn` handler (línea 541)
Mismo cambio: quitar campo `banUrl`, solo dejar `banFile`.

## Cambio 2: Productos - Cambiar a subida local

### `editProduct()` (línea 211)
**Cambiar el HTML de:**
```html
<div class="image-selector">
  <input type="hidden" id="prodImageUrl" value="${p.image || ''}">
  <div class="selected-image-preview" id="prodPreview" style="...">
    <img src="${p.image || ''}" alt="Imagen seleccionada">
    <button type="button" class="btn-remove-image" id="prodRemoveImg">×</button>
  </div>
  <button type="button" class="btn btn-secondary btn-sm" id="prodSelectImgBtn">📷 Seleccionar de Galería</button>
</div>
```

**A:**
```html
<div class="form-group">
  <label>Imagen del Producto</label>
  <input type="file" id="prodFile" accept="image/*" style="margin-bottom:8px">
  <div id="prodPreview" style="margin-top:8px;display:${p.image ? 'block' : 'none'}">
    <img src="${p.image || ''}" style="max-width:100%;max-height:200px;border-radius:8px">
  </div>
</div>
```

Y en el handler async, cambiar:
```javascript
const image = document.getElementById('prodImageUrl').value.trim();
```

**A:**
```javascript
const fileInput = document.getElementById('prodFile');
let image = '';
if (fileInput.files && fileInput.files[0]) {
  image = await readFileAsDataURL(fileInput.files[0]);
}
```

Y **eliminar** la línea:
```javascript
setupImageSelector('prodSelectImgBtn', 'prodRemoveImg', 'prodImageUrl', 'prodPreview');
```

Agregar preview en tiempo real:
```javascript
const prodFile = document.getElementById('prodFile');
const prodPreview = document.getElementById('prodPreview');
if (prodFile && prodPreview) {
  prodFile.addEventListener('change', () => {
    if (prodFile.files && prodFile.files[0]) {
      const reader = new FileReader();
      reader.onload = (e) => {
        prodPreview.querySelector('img').src = e.target.result;
        prodPreview.style.display = 'block';
      };
      reader.readAsDataURL(prodFile.files[0]);
    }
  });
}
```

### `showAddProduct()` (línea 270)
Mismo cambio: reemplazar image-selector con file input.
