# Plan: Activar Supabase

## Objetivo
Activar Supabase como fuente de datos para que los cambios del admin se compartan entre todos los usuarios y dispositivos.

## Cambio necesario

En `js/storage.js`, línea 119-120, cambiar:

```javascript
// ANTES
function useSupabase() {
  return false; // Forzar uso de LocalStorage
}

// DESPUÉS
function useSupabase() {
  return true; // Usar Supabase
}
```

## Archivo a modificar
- `js/storage.js`

## Verificación
1. Abrir admin panel
2. Agregar un producto
3. Abrir el sitio en otro navegador/dispositivo
4. Verificar que el producto aparece
