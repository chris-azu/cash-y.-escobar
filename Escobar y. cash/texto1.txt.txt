# PROYECTO NUEVO DESDE CERO - TIENDA DE ELECTRODOMÉSTICOS PROFESIONAL

## OBJETIVO

Crear una aplicación completamente nueva desde cero.

NO reutilizar la arquitectura actual.

El proyecto actual servirá únicamente como referencia visual y funcional.

Quiero una arquitectura profesional, escalable y lista para producción.

# TECNOLOGÍAS OBLIGATORIAS

Backend:
- Python
- Django 5+

Base de datos:
- PostgreSQL de Supabase

Frontend:
- Django Templates
- HTML
- CSS
- JavaScript

Almacenamiento de imágenes:
- Supabase Storage

# IMPORTANTE

NO utilizar:

- LocalStorage como almacenamiento principal.
- JSON como almacenamiento principal.

- Botones de prueba.
- CRUD falso.

# CONEXIÓN A SUPABASE

Necesito:

- Configuración correcta en settings.py.
- Variables de entorno en .env.
- Conexión estable.
- Migraciones funcionando.
